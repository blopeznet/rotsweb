# 🎸 Rise of the Shadows - Sitio Web Fan

Aplicación web moderna y responsiva fanpage para la banda **Rise of the Shadows**, desarrollada con **Vue 3** y **Vite**. Cuenta con una integración automatizada para mostrar las últimas publicaciones de Instagram de forma limpia y estilizada, utilizando una arquitectura Serverless.

---

## 🚀 Características

* ⚡ **Rendimiento optimizado:** Compilación ultra rápida e inicio instantáneo gracias a Vite.
* 🎨 **Estilo Metalero Custom:** Diseño oscuro integrado perfectamente con la estética de la banda, eliminando los iFrames rígidos de Facebook/Instagram.
* 🤖 **Scraper Serverless Integrado:** Consumo de las últimas publicaciones de las redes mediante una función backend en Node.js que se ejecuta en la nube de Vercel (`/api/get-socials.js`).
* 📦 **Caché Global:** Sistema de caché perimetral (Edge Network) de 12 horas para evitar bloqueos por parte de Meta y garantizar cargas instantáneas a los usuarios.
